# ISC-S1
## Glosario – Introducción a Sistemas de Cómputo

1. **Sistema de cómputo:** Conjunto de hardware, software, datos, usuarios y procedimientos que trabajan juntos para procesar información.  
2. **Hardware:** Componentes físicos de una computadora, como teclado, monitor, CPU y mouse.  
3. **Software:** Programas e instrucciones que indican al hardware qué hacer.  
4. **Dato:** Hecho o valor sin procesar que por sí solo no tiene significado.  
5. **Información:** Datos procesados y organizados que tienen significado y utilidad.  
6. **Entrada:** Datos que se introducen a un sistema para ser procesados.  
7. **Proceso:** Operaciones que transforman los datos de entrada.  
8. **Salida:** Resultado final del procesamiento de datos.  
9. **Almacenamiento:** Lugar donde se guardan datos e información para uso futuro.  
10. **Sistema operativo:** Software principal que administra el hardware y recursos del sistema.  
11. **Base de datos:** Conjunto organizado de datos almacenados electrónicamente.  
12. **Computación en la nube:** Uso de servicios de cómputo a través de internet.  
13. **Inteligencia artificial (IA):** Tecnología que permite a las máquinas simular inteligencia humana.  
14. **Ciberseguridad:** Protección de sistemas, redes y datos contra ataques digitales.  
15. **Usuario:** Persona que interactúa con el sistema de cómputo.  
16. **Red informática:** Conjunto de dispositivos conectados entre sí que comparten información y recursos mediante medios de comunicación.
